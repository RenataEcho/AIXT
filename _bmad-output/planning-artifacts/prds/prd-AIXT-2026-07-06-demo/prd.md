## 1. 愿景

**小提大作（融合升级版）** 是面向网文、剧本、短剧漫剧与微电影创作者的 **「AI 创作 + 内容/工作流/知识库交易 + 版权合作 + 收益分成分佣」一体化中台**。

用户在同一账户内完成：写小说 / 做漫剧视频 → 确权 / 挂牌 / 获授权 → 售卖 / 播放分成 / 工作流订阅 → 邀请分佣 → 提现开票。创作与变现不再割裂。

产品以 Demo 呈现的 **7 项主导航 + 5 项底部栏** 为工作台骨架：**大赏 · Agent · 文创 · 工具 · 创作 · 资产 · 结算**，吸收大作联盟的 SOP 流水线、WWDS 剧本评估、工具箱与共享 Agent，并在其上补齐商业闭环。

差异化主张：**「可复盘、可学习、可复用的创作过程」** 比单纯炫成片更有价值 —— 成片、工作流与 Agent 彼此引流，构成增长与变现飞轮。

## 2. 目标用户

### 2.1 Jobs To Be Done

- **个人网文/剧本创作者**：高效产出小说或可拍剧本，挂牌授权换取收益。
- **漫剧/短剧视频创作者**：用 AI 流水线把文本变成成片，降低制作门槛，靠售卖/播放分成变现。
- **新手创作者**：需要 SOP 分阶段引导，不必理解节点图也能产出一部作品。
- **熟练创作者/工作室**：需要自由节点画布 + 可复用工作流 + 团队调度质检。
- **版权方/IP 合作方**：登记确权、上架授权，靠授权分成变现，线上结算与开票。
- **变现型用户（分销/达人）**：靠邀请分佣获得收益，需要清晰的佣金、提现与发票能力。
- **平台运营方**：需要统一的确权、交易、结算、风控与数据看板。

### 2.2 非目标用户（v1）

- 纯知识付费/在线课程教学者 —— 知识库不含教程/方法论类内容（决策⑤）。
- 无创作意图、仅消费成片的纯观众 —— 大赏面向引流，非独立长视频平台。
- 需要私有化部署的企业 —— v1 为云端 SaaS。`[ASSUMPTION]`

### 2.3 关键用户旅程

- **UJ-1. 阿哲用 SOP 做出人生第一部漫剧**
  阿哲登录后从「创作」选影片 Tab 起项目，进入画布 **SOP 引导模式**，逐阶段确认选题 → 风格 → 剧本 → 分镜 → 人物 → 场景 → 合成视频，消耗爆米花，失败可重试断点续跑。导出成片后挂到大赏。**价值落点**：第一次不看教程就完整产出一部作品。

- **UJ-2. 林姐把畅销小说确权后授权给漫剧作者**
  林姐在写作模块完成《凡人修仙诀》后，一键到文创**登记确权**并挂牌。漫剧作者小鹿在文创提交**授权申请**，审核通过后基于 IP 制作。收益在**结算中心**查看并**提现**。**价值落点**：创作与授权收益同一平台闭环。

- **UJ-3. 编剧小陈用工具评估剧本后上架文创库**
  小陈在「工具」使用**剧本评估**，粘贴剧本获得 WWDS 六维报告，提交审核后在文创**剧本版权**库展示；承制方浏览详情、**申领剧本**走商务对接。**价值落点**：评估—上架—申领链路可运营。

- **UJ-4. 小鹿把工作流上架市场，靠订阅与分佣变现**
  小鹿沉淀「小说转漫剧」工作流，登记产权后上架**交易市场**（Demo 隐藏页，目标态），设为订阅制；下级消费产生邀请分佣，统一在结算提现。**价值落点**：「怎么做」本身变成可持续售卖的资产。

## 4. 产品链路

### 4.1 顶层商业链路

```
游客引流（大赏 / 教程 / 体验）
    ↓ 注册登录
创作生产（写作 🌽 | 画布/SOP 🍿 | 工具链辅助）
    ↓
资产沉淀（角色/场景/Agent/提示词）
    ↓
确权挂牌（文创 · 小说/剧本版权库）
    ↓
交易变现（市场 · 售卖/订阅/授权分成）[目标态]
    ↓
曝光复盘（大赏 · 成片反向查看创作过程）
    ↓
结算闭环（结算中心 · 版权收益 + 爆米花 + 提现/发票）
    ↓
增长裂变（邀请码 · 邀请分佣 → 同账户结算）
```

### 4.2 创作双轨链路

**轨道 A · 写作（米花）**
```
创作(写作Tab) → 写作编辑器(章节目录|正文|AI助手)
  → 导出全文 → 一键确权挂牌 → 文创(小说版权)
```

**轨道 B · 画布/SOP（爆米花）**
```
创作(影片Tab) → 画布
  ├─ SOP: 选题(类型+众筹+题材) → 风格 → 剧本 → 分镜 → 人物 → 场景 → 合成视频
  └─ 自由画布: 节点编排(小说转剧本→分镜→生图→图生视频→调度台)
  → 导出成片 → 大赏曝光 / 文创确权
```

**轨道 C · 工具辅助链**
```
工具箱
  ├─ 剧本评估(全页) → WWDS报告 → 提交审核 → 文创剧本库
  ├─ 剧本拆解(全页) → 分集脚本 → 复制/下游制作
  ├─ 视频结构提取(弹窗) → 细纲 → 「用细纲写剧本」→ 写作
  └─ 提示词/音频/资产/IP提取(弹窗) → 结果块 → 提交审核
```

## 5. 产品生态

### 5.1 双货币计费体系

| 货币 | 符号 | 消耗场景 | Demo 可见位置 |

| 维度 | 取值/规则 |

### 5.4 结算闭环

```
收入来源
  → 结算中心(settlement) · 概览三卡(累计/已提现/可提现)
    ├─ 版权结算 Tab · 四维 breakdown + 明细列表
    ├─ 爆米花 Tab · 余额构成 + 兑换/购买 + 流水
    └─ 提现记录 Tab · 完整表格(手续费/到账/状态)
  → 扩展(data 页) · 看板/结算明细/提现/发票
```

**本期新建（决策④）**：提现 + 发票；邀请分佣规则沿用现有产品，不重做。

### 5.5 版权/文创生态

- **小说版权库**：10+ 真实风格书目；独家/非独家/分成/询价；5 维筛选 + 排序；hover 商务信息；询价弹窗。
- **剧本版权库**：WWDS 评估数据；六维雷达 + 改编潜力 + 受众风险 + 完整报告；申领走商务二维码。
- **爆榜**：ranking mock 数据存在，Tab 入口 Demo 未开放（orphan UI）。

### 5.6 社区生态（大赏）

- Hero 轮播 + 三维搜索（类型/作者/标题）+ 海报网格。
- 社交元数据：作者/点赞/收藏/分享（mock）。
- 目标态：五种互动（赞/藏/评/享/关）、评论三泳道、从成片反向复盘创作过程。

### 5.7 播放奖励阶梯

500 → 1 万 → 5 万 → 10 万（封顶）；数据来源：用户自报 + 第三方回传（协议待定，决策⑥）。

### 6.2 页面功能详述

#### 6.2.1 大赏（showcase）

**页面目标**：IP 与影片展示广场，引流与社区互动入口。

**功能区块**：
- Hero 轮播（`showcaseHeroSlides`）
- 三维搜索栏：类型 / 作者 / 标题
- 海报作品网格（`posterWorks`，含封面、作者、点赞/收藏/分享等）

**交互**：Hero 切换；搜索过滤；卡片点击 `[ASSUMPTION: 跳转作品详情/复盘页，Demo 为展示态]`

**需求要点**：
- 展示 IP 信息与影片成片
- 登录态互动；未登录弹登录浮层后补执行
- 目标态：反向复盘创作过程 → Agent/市场引流

---

#### 6.2.2 Agent（prompts）

**页面目标**：共享 Agent / 提示词库，供封面/视频/写作/剧本阶段一键调用。

**功能区块**：
- 一级 Tab：封面 | 视频 | 写作 | 剧本
- 关键字搜索 + 二级题材 chips（`promptFamilies`）
- 卡片流：code、title、genre、hot、tags、paid、封面预览

**交互**：选卡片 → `promptDetailDialog`（完整 prompt + 复制）；切换 mode 重置 subKey。

**数据来源**：`PROMPT_LIBRARY_DATA`（cover/video/writing/script 四套库）。

---

#### 6.2.3 文创（copyright）

**页面目标**：小说与剧本版权交易、评估、商务对接。

**Landing 态**：
- Hero 轮播（`cpBanners`）
- 双入口卡片：编剧投稿 / 作家发布小说
- 主 Tab：**小说版权** | **剧本版权**
- 小说：搜索 + 5 维筛选（偏好/篇幅/类型/完结/版权方）+ 排序 + 宫格 + hover 商务信息 + 询价弹窗
- 剧本：搜索 + 宫格 → 点击进入详情

**剧本详情子页**（`cpScriptDetailId` 有值）：
- Tab：概览 | 改编潜力 | 受众与风险 | 完整报告
- WWDS 六维：w/i/h/g/r/v + 评级 + 红线扫描
- 操作：申领剧本 → `cpScriptClaimDialog`（商务二维码）

**数据模型**：
- 小说：`cpBooks`（title, author, exclusiveType, cooperation[], wordCount…）
- 剧本：`cpScripts` + `WWDS_SCRIPT_DATA.details[id]`

---

#### 6.2.4 工具（tools + 子页）

**页面目标**：创作前/中期的 AI 辅助工具链。

**工具箱 7 项**：

| ID | 名称 | 形态 | 输入 | 输出 |
|----|------|------|------|------|
| script-eval | 剧本评估 | 全页 | 粘贴剧本 | WWDS 六维报告 |

## 9. 非功能需求（NFR）

- **性能**：AI 生成异步 + 进度反馈；画布大工程流畅平移。
- **计费一致性**：`getCost` 预估与实际扣费一致，幂等与对账。
- **资金安全**：提现/发票/分成需审核流、审计、二次校验。
- **风控**：播放分成、邀请分佣、自报链接防作弊。
- **权限**：工作台鉴权；文创/资产/结算按角色；指挥舱 Pro 门槛。
- **数据一致性（决策②）**：个人创作与指挥舱同数据源双视角。
- **国际化**：21 语言 + 暗/亮主题；Demo 默认 dark。
- **UI 约束**：Vue3 + Element Plus；`/api/fc/` + `/api/synapse/` 双通道；不引入第二套设计语言。
