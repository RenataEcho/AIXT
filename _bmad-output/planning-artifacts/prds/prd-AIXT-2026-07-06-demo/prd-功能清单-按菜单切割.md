# 第一部分 · 产品整体说明

## 1. 产品定位

**小提大作（融合升级版）** 是面向网文、剧本、短剧漫剧、微电影创作者的 **AI 创作 + 版权交易 + 收益结算** 一体化中台。用户在同一账户内完成「创作 → 确权 → 交易/授权 → 曝光 → 结算 → 裂变」全链路。

**差异化**：「可复盘、可学习、可复用的创作过程」—— 成片、Agent、工作流彼此引流，构成增长与变现飞轮。

## 4. 生产链路

### 4.1 顶层生产链路

```
游客 / 新用户
  → 大赏浏览 / 教程学习
  → 注册登录
  → 【创作】起项目
       ├─ 轨道A 写作(🌽) → 导出 → 文创确权
       ├─ 轨道B 画布/SOP(🍿) → 成片 → 大赏 / 文创
       └─ 轨道C 工具辅助 → 评估/拆解/提取 → 写作或画布
  → 【资产】沉淀角色/场景
  → 【Agent】调用共享模板
  → 【文创】挂牌 / 申领 / 授权
  → 【大赏】曝光 / 复盘
  → 【结算】收益 / 提现
  → 【邀请】裂变拉新
```

### 4.2 轨道 A · 写作链路（🌽 米花）

```
创作(写作Tab) → 写作编辑器
  作品设定 → 章节目录 → 正文编辑 → AI助手(大纲/细纲/正文)
  → 导出全文 → 一键确权挂牌 → 文创(小说版权库)
```

**使用对象**：网文作者、文本创作者、需产出可授权小说 IP 的版权方。

### 4.3 轨道 B · 画布/SOP 链路（🍿 爆米花）

```
创作(影片Tab) → 画布
  【SOP 引导模式】
    选题(类型+众筹+题材) → 风格 → 剧本 → 分镜 → 人物 → 场景 → 合成视频
  【自由画布模式】（与 SOP 共享项目数据）
    节点编排：小说转剧本 → 分镜 → 生图 → 图生视频 → 场景调度
  → 导出成片 → 大赏曝光 / 文创确权
```

**使用对象**：漫剧/短剧创作者、新手（SOP）、熟练用户/工作室（自由画布）。

### 4.4 轨道 C · 工具辅助链路

```
工具箱
  ├─ 剧本评估 → WWDS报告 → 审核 → 文创剧本库
  ├─ 剧本拆解 → 分集脚本 → 下游制作
  ├─ 视频结构提取 → 细纲 → 跳转写作
  └─ 链接提取类(5项) → 结果块 → 审核 / 创作引用
```

**使用对象**：编剧、承制方、需从外部素材快速起稿的创作者。

### 4.5 版权→变现链路

```
写作/画布产出
  → 文创登记(小说/剧本)
  → 承制方浏览 / 筛选 / 申领(剧本) / 询价(小说)
  → 授权审核 → 制作 → 播放/售卖
  → 版权分成 → 结算中心 → 提现
```

### 4.6 团队生产链路（目标态 · 指挥舱）

个人 SOP/画布产物 **自动映射** G0–G9 Gate 看板，一份数据双视角。

| 创作阶段 | Gate | 负责部门 |
|----------|------|----------|
| 选题/立项 | G0/G1 | PMO |
| 剧本 | G2 | 编剧中心 |
| 分镜 | G3 | 分镜导演 |
| 人物/场景 | G4 | 资产中心 |
| 分镜图/画布 | G5 | AI 图像 |
| 视频 | G6 | AI 视频 |
| 成片 | G7 | 剪辑声音 |
| 审核 | G8 | QC 合规 |
| 复盘 | G9 | PMO |
